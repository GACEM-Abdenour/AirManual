Receive e-mail notifications for a filter

    

To specify that you would like to receive an e-mail notification whenever a publication which matches one of your filters is uploaded, you have to first select the (saved) filter by clicking on it. Then click on the "Notify" button inside the blue panel on the left. If the operation was successful, you will receive a confirmation message.

 

Filters which are "active" for notifications are marked with an orange colour. You can have multiple filters marked as active for notifications at the same time.

 

To specify that you would like to stop receiving notifications for a particular filter, select the filter and click on the "Notify" button again. If the operation was successful, you will receive a confirmation message.

 

If you would like to stop receiving any notifications from the Safety Publications Tool application in general, instead of marking all your filters as inactive, you can click on the "My details" button on the top toolbar, provide your password and set the global setting "Get notification emails" to "Unsubscribed".

 

Warning: If you create a filter which contains an ending issue date in the past, you will not receive notifications for this filter.

 

Warning: The global setting "Get notification emails" that you specified when you registered for the application, overrides the preferences that you specify in the advanced search/filters page. If that global setting is set to "Subscribed", you will be notified of all new items published (if you don’t have notifications set for a personal filter), or you will be notified of all new items published which belong to your notified personal filter(s). If your global setting is “Unsubscribed”, you will not receive any notification, even if you save and notify a personal filter. To check or change this setting, please click on the "My details" button on the top toolbar.

 

Receive notifications only for custom filter

To receive notifications only for your custom filter, please log in, go to the tab Advanced Search and deactivate notifications for the generic filter (show all) by clicking on that filer, deselecting all categories (AD, EAD, etc.) and clicking on Notify.

After that, please refresh the page. You will notice that the color of the filter has changed.

Please note that filters with active notifications are marked with an orange color.
