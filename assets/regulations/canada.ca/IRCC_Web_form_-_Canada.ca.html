IRCC web form: Contact us online

Contact us through our web form to get help with your application.

Before you contact us

  • Check your application status
  • Check processing times
  • Find a form or application package
  • Ask for a refund
  • Get technical help
If you haven’t heard from us

If you submitted an application and haven’t heard from us recently, it means we’re still processing your application. You don’t need to do anything else. If we need more information, we’ll contact you. Submitting another web form won’t speed things up.

After you submit a web form, you’ll get an acknowledgement of receipt (AOR) to confirm we received your form.

How we can and can’t help you through the web form

We can help you

  • get answers to general questions about your application or case
  • check your application status
    • Our agents don’t have more details than what’s available online.
  • find information on our website

We can’t

  • advise you on which program to apply for
  • speed up your application’s processing time (unless you’re already eligible for urgent processing)
  • make a decision on your application

Help with filling out the web form

Application number and unique client identifier (UCI)

If you’ve never sent an application

You won’t have an application number or unique client identifier (UCI) yet. You don’t need one to

  • report technical issues
  • ask about our programs

How to fill out the web form:

  • In the application number field, enter 0000000.
  • In the UCI field, enter 1111111111.

If you have an application in progress

  • How to find your application number
  • How to find your UCI
“Ask about or update your application” section

Avoid these common mistakes when filling out the “Ask about or update your application” section of the web form.

Temporary residence (visit, study, work)

When you’re asked “What did you apply for?”

  •  select “Temporary Residence” if you applied for
    • an electronic travel authorization (eTA)
    • a visitor visa
    • a work permit
    • a study permit
    • a visitor record
  •  don’t select “Status document”

Visitor visas

If you’ve applied for a visitor visa, when you’re asked “What application did you submit?”

  •  select “Temporary Resident Visa”
  •  don’t select
    • “Visitor Record”
    • “Temporary Resident Permit”
    • “Electronic Travel Authorization”

Extensions for biometrics or medical exams

If you need extra time to give your biometrics or get a medical exam, when you’re asked “What would you like to do?”

  •  select “Other”
  •  don’t select “Schedule or change the date or time of your appointment, interview or test”
