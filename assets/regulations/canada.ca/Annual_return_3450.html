Annual return

Every federal corporation has to file its annual return with Corporations Canada every year. Filing an annual return avoids the potential administrative dissolution of a business or not-for-profit corporation.

A corporation key is mandatory to file an annual return and individuals with significant control information for businesses incorporated under the CBCA.

Eligibility

Federal business and not-for-profit corporations can file their annual return online.

Search for a federal corporation to find out if your annual return is due, if you're not sure.

How much it costs

An annual return costs:

  • $12 (online)
